# -ˋˏ✄┈┈┈┈virtual corkboard┈┈┈┈

Виртуальная пробковая доска-планер для создания и размещения стикеров.

## Описание

Планер позволяет пользователю добавлять текстовые заметки в виде стикеров, выбирая их цвет и форму при создании, позволяет перемещать стикеры по доске, удалять или сохранять их. Также пользователь при необходимости может распечатать свою доску.

## Управление

| Метод                                              | Описание                               |
| -------------------------------------------------- | -------------------------------------  |
| ⊂⊃ Канцелярская кнопка                            | Открытие главного меню                 |
| ⊹ Добавление стикера                              | Открытие формы создания стикера        |
| ✉ Сохранение                                      | Сохранение стикеров                    |
| ✁ Сброс                                           | Очищение доски от всех стикеров        |
| ⎙ Печать                                          | Скриншот экрана для последующей печати |
| ▿ Удерживание стикера правой кнопкой мыши         | Перемещение стикера                    |
| ◉ Нажатие на стикер                               | Приближение стикера                    |
| ☓ Нажатие на красный кружок на стикере            | Удаление стикера                       |

**Не забывайте про сохранение после глобального очищения доски!!!**

## Требования

☆ Docker и Docker compose.

## Запуск проекта

**Команды для запуска в терминале C использованием Docker:**

```bash
git clone https://github.com/qazwsxill/Virtual_corkboard
cd Virtual_corkboard
docker compose up --build
```
Адрес для открытия в браузере http://localhost:8080

**Команды для запуска в терминале БЕЗ использования Docker:**

`Запуск backend:`

```bash
git clone https://github.com/qazwsxill/Virtual_corkboard
cd Virtual_corkboard/backend
npm install
node server.js
```
`Запуск frontend:`

Команды необходимо вводить в новом (втором) терминале.

```bash
cd Virtual_corkboard/frontend
npx live-server
```
